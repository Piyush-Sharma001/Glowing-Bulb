💡 Glowing Bulb Project

A simple and interactive Glowing Bulb project built using HTML5, CSS3, and JavaScript for learning DOM Manipulation concepts.

This project demonstrates how JavaScript can dynamically interact with webpage elements using event handling and timer functions.

🚀 Project Overview

The Glowing Bulb project allows users to turn a bulb ON/OFF & Dipper mode with interactive effects and animations.

The main goal of this project was to practice:

DOM Manipulation
Event Handling
Timer Functions
Dynamic UI Updates

✨ Features
Bulb ON/OFF Functionality 💡
Interactive Button Controls
Glowing Effect Animation
Event Handling using onclick
Timer Functions using setInterval()
Beginner-Friendly JavaScript Logic
